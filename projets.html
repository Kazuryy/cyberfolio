Mes projets réalisés :

Site e-commerce

Une plateforme de commerce électronique avec des fonctionnalités comme l'inscription des utilisateurs, un panier d'achat, un système de paiement sécurisé, et une gestion des produits.

Techonologies utilisées : Front-end : React, Vue.js, ou Angular. Back-end : Node.js avec Express, Django, ou Ruby on Rails. Base de données : MongoDB, PostgreSQL, ou MySQL. Extras : Stripe/PayPal API pour les paiements.

Applications de gestion de tâches

Une application de type "to-do list" qui permet aux utilisateurs de créer, modifier, supprimer et organiser leurs tâches.

Techonologies utilisées : Front-end : React, Vue.js, ou Svelte pour l'interface utilisateur. Back-end : Node.js avec Express, Flask, ou Django pour la gestion des données. Base de données : MongoDB, Firebase Firestore, ou PostgreSQL.

Réseau social minimaliste

Une plateforme où les utilisateurs peuvent s'inscrire, publier des messages, suivre d'autres utilisateurs, et interagir via des commentaires et des likes.

Techonologies utilisées : Front-end : Next.js ou Nuxt.js pour les performances. Back-end : GraphQL avec Apollo Server ou REST API. Extras : WebSockets pour le chat en temps réel.

Dashboard interactif

Un tableau de bord avec des visualisations de données (graphiques, tables interactives) pour un cas d'usage particulier (finances, logistique, marketing).

Techonologies utilisées : Front-end : D3.js, Chart.js, ou Recharts. Back-end : Flask ou Express.js. Extras : Intégration avec des APIs tierces pour des données en direct (API météo, finance, etc.).

Jeu web multijoueur simple

Un petit jeu interactif en ligne, comme un jeu de cartes ou un quiz.

Techonologies utilisées : Front-end : React ou Vue.js. Back-end : Node.js avec WebSockets.
